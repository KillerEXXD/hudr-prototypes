# HUDR Lab Deep Prototype V2

Deep clickable prototype (React + Vite) for HUDR-like tournament intelligence UX.

## Run locally
```bash
npm install
npm run dev
```

## Included flows
- Onboarding / search / tournament landing
- Tournament overview + timeline
- Hands explorer + hand detail + replay center
- Players + player detail + compare
- AI studio + AI answer + evidence chain
- Highlights
- Report builder + exploit report
- Watchlist + notes
- Settings

Uses hash routing so all links work in static/local mode.
